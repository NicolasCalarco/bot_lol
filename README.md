#bot lolcito

Para su correcto funcionamiento, deberan crear la carpeta config. Y dentro de ella el archivo auth.py.

Este archivo debera contener lo siguiente:

```
token_telegram = ''
token_riot = ''
db_server = ''
db_database = ''
db_username = ''
db_password = ''
```

Y al finalizar, deberan ejecutar el archivo bot.py
